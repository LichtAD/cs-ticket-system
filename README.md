1. What is JSX, and why is it used?
=> JSX is a syntax extension for JavaScript which is used with React to write HTML like codes.

2. What is the difference between State and Props?
=> Props are read-only data passed down from parent to child components and state is internal data managed within a component which is similar to local variables.

3. What is the useState hook, and how does it work?
=> useState allows user to add state variables to functional components and give them the ability to remember data across re-renders. It returns an array with the current state value and a setter function which triggers a component re-render to update the UI.

4. How can you share state between components in React?
=> We can share state between components in React by passing down props.

5, How is event handling done in React?
=>  Events are handled by passing callback functions like onClick as props to JSX elements.
